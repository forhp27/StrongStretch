How This Works

  • This app uses Petite Vue for reactive UI components without a build step
  • Tasks are stored in Firebase Realtime Database and sync across devices
  • Add new tasks using the input field, mark them complete, edit or delete them
  • All changes are automatically saved to Firebase in real-time
  • To set up your own Firebase project, see instructions below
